Bem-vindo ao ASFAR

Sistema para análise de solo e recomendações agronômicas

Atualmente esse sistema está projetado para trabalhar com a correção da acidez do solo

Para realizar uma nova análise de solo, siga as instruções:

Com o ASFAR você pode armazenar todos os seus laudos de análises de solo em um único lugar de forma organizada;

1º Cadastre um novo clienteaqui;
2º Acesse o cliente que você cadastrou;
3º Selecione o botão de ação principal (canto direito inferior da tela) para cadastrar um novo talhão à esse cliente;
4º Acesse o talhão que você cadastrou em seu cliente;
5º Selecione o botão de ação principal para iniciar uma nova análise de solo e selecione as configuraçõe (se houver) com base na cultura agrícola que desejar.

As análises de solo podem ser facilitadas por configurações que vão guiar você atravéz de intruções, para configurar o sistema, siga as intruções à baixo.

Para configurar o sistema, siga as instruções:

As configurações permitem que o sistema sempre se mantenha atualizado com o ROLAS ou outras pesquisas agrícolas, dessa forma você não vair perder seus trabalhos por um sistema deteriorado;

1º Cadastre uma nova configuraçãoaqui;
2º Acesse a configuração que você cadastrou;
3º Selecione o botão de ação principal (canto direito inferior da tela) para cadastrar um novo pacote de configuração para a correção da acidez do solo;
4º Crie uma nova análise de solo e selecione a configuração e os pacotes que a compoem para facilitar seu trabalho.

Pacotes de configurações são partes da configuração que representam intruções para a correção do solo para uma cultura especifica ou grupos de culturas.
